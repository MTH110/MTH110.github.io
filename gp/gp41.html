Guided Practice 4.1: Quadratic Functions and Parabolas

Overview

In this section we're going to find that not all data sets follow a linear pattern. Some follow an "up and then down" or "down and then up" pattern, so a line won't fit the data -- but a curve might. We introduce a new kind of function called a quadratic function whose graph is a shape called a parabola that will fit data with this up-and-down or down-and-up patter well.

Learning objectives

Basic objectives: Each student is responsible for gaining proficiency with each of these tasks prior to engaging in class discussions, through the use of the learning resources (below) and through the working of exercises (also below).

  • Define the terms quadratic function and parabola.
  • Explain the differences between the standard form of a quadratic function and the vertex form of a quadratic function.
  • Given the formula for a function, determine whether the function is linear, quadratic, or neither.
  • Given the graph of a quadratic function, determine the following: the coordinates of its vertex; the values for which the function is increasing; the values for which the function is decreasing; the horizontal intercepts; the vertical intercept; and values of the function (for example, \(f(4)\)).

Advanced objectives: The following objectives are the subject of class discussion and further work; they should be mastered by each student during and following class discussions.

  • Given a table of data that have a quadratic pattern, find the horizontal and vertical intercepts of a quadratic function that fits the data.
  • Given a table of data that have a quadratic pattern, find the vertex of a quadratic function that fits the data and determine whether the vertex is a maximum point or a minimum point.
  • Use a quadratic function (given as a data table, graph, or formula) that models a real-life application and use the function to answer questions about the application.
